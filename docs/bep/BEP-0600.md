## Protocolo FL

```
Round N:
  1. FLCoordinator envĂ­a global_weights a clientes activos
  2. Cada cliente: entrena localmente 1 epoch con sus datos
  3. Cada cliente: llama run_round({client_id: weight_delta})

  5. Convergencia si L2(global_prev, global_curr) < 1e-4
  6. Repeat hasta convergencia o N rounds agotados
```
