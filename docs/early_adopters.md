# Programa Early Adopters — BESSAI Edge Gateway

> **Última actualización**: Marzo 2026 · **Estado**: 🟢 Activo — aceptando postulaciones

---

## ¿Qué es el programa Early Adopters?

El programa **BESSAI Early Adopters** conecta al equipo core del proyecto con organizaciones y profesionales que quieren ser los primeros en usar el gateway en instalaciones reales y contribuir a dar forma al producto con feedback directo.

No se trata de un simple beta test — es una **alianza técnica** donde ambas partes ganan.

---

## Beneficios para Early Adopters

### Para organizaciones
- 🛡️ **Soporte técnico prioritario** directamente con el equipo BESSAI (canal Slack privado, respuesta < 24h en días hábiles)
- 📣 **Mención prominente** en el README, documentación y página web (con tu logo, si lo autorizas)
- 🚀 **Acceso anticipado** a versiones beta con nuevas features (antes del release público)
- 🗳️ **Voz en el roadmap**: tus casos de uso influyen directamente en las prioridades de desarrollo
- 📊 **Dashboard analítico compartido**: acceso a métricas de rendimiento agrupadas de todos los Early Adopters (anónimas)

### Para investigadores / estudiantes
- Mentoría directa del equipo core
- Acceso a datasets históricos CMg Chile 2023–2025 (previa NDA simple)
- Co-autoría en publicaciones técnicas derivadas del trabajo conjunto
- Carta de colaboración institucional para postulaciones FONDECYT / Anid

## ¿A quién va dirigido?

Buscamos Early Adopters en estas categorías:

| Categoría | Perfil ideal |
|---|---|
| **Integrador de sistemas** | Instala BESS + solar en proyectos residenciales o C&I. Quiere usar BESSAI como plataforma de control y monitoreo. |
| **Operador de red / utilities** | Gestiona múltiples puntos de almacenamiento. Interesado en VPP y respuesta de frecuencia. |
| **Empresa de energía** | Desarrollador de proyectos solar + storage en Chile o Latam. Quiere optimización económica con DRL. |
| **Investigador / academia** | Universidad o centro de investigación desarrollando trabajos sobre almacenamiento, DRL o mercados eléctricos. |
| **Fabricante de hardware** | Fabricante de inversores, BMS o sistemas BESS interesado en certificación BESSAI-Ready. |

---

## Criterios de elegibilidad

Para ser aceptado como Early Adopter, se requiere:

- [ ] Hardware BESS disponible (real o simulado en entorno controlado)
- [ ] Capacidad técnica para instalar y configurar el gateway (Python, Docker, basic networking)
- [ ] Disponibilidad para proporcionar feedback estructurado mínimo 1 vez al mes
- [ ] Acuerdo de no utilizar el proyecto en entornos críticos sin las debidas precauciones y validaciones locales

> ⚠️ BESSAI Edge Gateway es un proyecto **open source en desarrollo activo**. El programa Early Adopters implica aceptar las condiciones de la [Licencia Apache 2.0](../LICENSE) y el [Código de Conducta](../CODE_OF_CONDUCT.md).

## Cómo postular

### Opción A: GitHub Discussion (recomendado)

1. Ve a [GitHub Discussions → Early Adopters](https://github.com/bess-solutions/open-bess-edge/discussions/new?category=early-adopters)
2. Usa el template **"Early Adopter Introduction"** — te guiará con preguntas estructuradas sobre tu hardware, caso de uso y mercado.

> 💡 El template está en `.github/DISCUSSION_TEMPLATE/early_adopter_intro.yml` — cubre organización, hardware, mercado, experiencia técnica y caso de uso.

### Opción B: Email directo

Escríbenos a `early-adopters@bessai.io` describiendo brevemente tu hardware y caso de uso.

---

## Proceso de selección

1. **Postulación** vía GitHub Discussion o email
2. **Revisión** por el equipo core (plazo: 5 días hábiles)
3. **Call introductorio** (30 min): entender tu caso de uso y setup técnico
4. **Onboarding**: acceso al canal Slack privado + guía de instalación personalizada
5. **Primer feedback** al mes de uso

## Preguntas frecuentes

**¿Tiene costo?**  
No. El programa es gratuito y siempre lo será. BESSAI es open source.

**¿Puedo salir del programa cuando quiera?**  
Sí, en cualquier momento y sin compromisos.

**¿Tengo que compartir mis datos de producción?**  
No. Nunca pedimos acceso a datos operativos reales. El feedback es siempre voluntario y puede ser anónimo.

**¿El programa está disponible fuera de Chile?**  
Sí. Bienvenidos de toda Latinoamérica, España, y el mundo.
